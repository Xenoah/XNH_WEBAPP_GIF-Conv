# Pic2GIF — 仕様書・作業メモ

## 概要

複数の画像ファイルをブラウザ上でGIFアニメーションに変換するWebアプリ。
サーバー不要・完全クライアントサイド動作。GitHub Pages で公開可能な単一HTMLファイル構成。

---

## 使用技術

| ライブラリ / ツール | バージョン | 用途 |
|---|---|---|
| React | 18.2.0 (CDN) | UIフレームワーク |
| Babel Standalone | 7.23.5 (CDN) | JSX トランスパイル |
| Tailwind CSS | latest (CDN) | スタイリング |
| gif.js | 0.2.0 (CDN) | GIFエンコーダー |
| Inter (Google Fonts) | — | フォント |

## 機能一覧

### 画像アップロード
- ドラッグ＆ドロップでファイルを追加
- クリックでファイル選択ダイアログ
- 複数ファイル同時読み込み対応
- 対応形式: PNG / JPG / WebP / GIF

### フレーム管理
- サムネイルグリッド表示（レスポンシブ: 4〜10列）
- フレームの左右移動（順序変更）
- フレーム個別削除
- 全フレーム一括クリア
- フレームクリックでプレビュー選択

### 設定パネル（スライダー）
| 設定項目 | 範囲 | デフォルト | 説明 |
|---|---|---|---|
| フレーム遅延 | 10〜3000ms | 100ms | 各フレームの表示時間 |
| ループ回数 | 0〜10 | 0 (∞) | 0 = 無限ループ |
| 出力幅 | 64〜1200px | 480px | GIF出力幅（高さは自動計算） |
| 品質 | 1〜10 | 7 | 10が最高品質 |

### プレビューパネル
- 選択フレームの画像表示
- 再生 / 一時停止
- フレーム単位移動（前・次）
- 最初/最後フレームへのジャンプ
- 再生速度は「フレーム遅延」設定と連動

### GIF生成・ダウンロード
- 「GIF 生成」ボタンで非同期エンコード
- エンコード進捗バー表示（0〜100%）
- キャンセルボタンで途中停止可能
- 生成完了後: ファイルサイズ表示、GIFプレビュー表示
- 「ダウンロード」ボタンで `animation.gif` として保存

### UI
- ダークモード対応（システム設定を自動検出、手動切替可）
- レスポンシブデザイン（モバイル〜デスクトップ）

## 技術メモ

### gif.js の Web Worker 問題と対処
- gif.js は GIF エンコードに Web Worker を使用するが、Worker はクロスオリジンURLから直接ロードできない
- **対処**: `fetch()` で `gif.worker.js` を取得し、`Blob URL` に変換して使用
  ```javascript
  fetch('https://cdn.jsdelivr.net/npm/gif.js@0.2.0/dist/gif.worker.js')
    .then(r => r.blob())
    .then(blob => { blobUrlRef.current = URL.createObjectURL(blob); });
  ```

### 出力サイズ計算
- 出力幅はユーザー設定値 `settings.width`
- 出力高さは **1枚目フレームのアスペクト比** から自動計算
  ```
  outH = floor(firstFrame.naturalHeight * outW / firstFrame.naturalWidth)
  ```

### フレームスケーリング（レターボックス）
- 各フレームを出力サイズにスケーリング（アスペクト比を維持）
- 余白は白（`#ffffff`）で埋める
- 異なるアスペクト比の画像を混在させてもGIFは正常に生成される

### 品質設定の変換
- UIスライダー: 1〜10（10が高品質）
- gif.js の `quality` パラメータ: 1〜20（**1が高品質**、値が小さいほど高品質）
- 変換式: `gifQuality = max(1, round(21 - uiQuality * 2))`
  - UI=10 → gifQuality=1 (最高品質)
  - UI=1  → gifQuality=19 (最速)

### メモリ管理
- 生成済み Blob URL は次回生成時に `URL.revokeObjectURL()` で解放
- Worker Blob URL はコンポーネントアンマウント時に解放
